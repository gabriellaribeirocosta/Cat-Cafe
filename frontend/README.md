# Cat Cafe | Front-end

## Getting Started

### Prerequisites

Ensure you have the following installed on your machine:

- Node.js
- npm

### Installation

Clone the repository and install the dependencies:

```bash
cd cat-cafe/frontend
npm install

# Start the development server
npm run dev
```

## File Structure

