# Cat Cafe API

## Installation

Change into the backend repository and install the dependencies:

```bash
cd cat-cafe/backend
npm install

# Start the development server
npm run start:dev
```

## File Structure

The backend follows the Model-View-Controller (MVC) framework.

```
./
|-- src/
    |-- controllers/
    |-- database/
    |-- middlewares/
    |-- models/
    |-- routes/
    app.js
```
